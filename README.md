# Thanksy

## Setup

to install dependencies

```sh
npm install
```

## Running

to run application with hot reloading

```sh
npm start
```

then visit `http://localhost:4004/`

## Testing

to run unit tests

```sh
npm run test:unit
```

to run acceptance tests (e2e)

```sh
npm run test:e2e
```

to run all tests

```sh
npm test
```
